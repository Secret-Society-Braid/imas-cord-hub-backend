## Description

ImasCordHub Backend is an Free Open Source Software (FOSS) and represents the neccessary data for the ImasCordHub Frontend app. 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
